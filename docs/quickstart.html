Aloha Editor Quickstart

To use Aloha Editor in your application you will just need to load the minified JavaScript file

and create an editable region in the page, by using the aloha() function

HTML Code

<div class="aloha-editable">
	<h2>This content will be editable...</h2>
	<p>
		if you press the 
		<button class="btn btn-sm" onclick="executeExample('example-aloha-func')"
		contentEditable="false">Execute Example</button>* button.
	</p>
	<p><small>*) Yes, you can even press the button in the example</small></p>
</div>

Result

This content will be editable...

if you press the * button.

*) Yes, you can even press the button in the example
